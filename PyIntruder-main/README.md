## Description

PyIntruder is a powerful intruder tool built in Python that allows you
to: - Perform Wordlist Attacks - Perform Number-based Attacks - Perform
BruteForce Attacks - Encode payloads using Base64, Hex, or ASCII
Numbers - Use URL Encoding

## Features

### Brute Force

-   Wordlist
-   Numbers
-   Characters

### Encoding Support

-   Base64
-   Hex
-   ASCII Numbers
-   URL Encoding
